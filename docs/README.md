各コンペティションのドキュメントは、`competitions/[competition]/docs/` に配置されています。

- `OVERVIEW.md`: Kaggleのoverviewページの内容
- `DATA.md`: Kaggleのdataページの内容
- `experiment_workflow.md`: 実験の進め方
